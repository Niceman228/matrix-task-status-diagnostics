# Метод дефицита — веб-приложение

Интерактивный интерфейс для оценки статуса проектных задач по матрице связности. Приложение помогает быстро классифицировать постановку как расчётную, оптимизационную или неразрешимую, а также проверить корректность пар параметров и наличие информационных связей между операциями.

## Возможности
- наглядное редактирование бинарной матрицы участия операций в параметрах;
- три режима анализа (статус задачи, пара \(I, T\), информационная связь);
- автоматический пересчёт дефицитов и подсветка строк без влияния на результат;
- детализированный отчёт с ASCII-таблицей матрицы и выводами на русском языке;
- поддержка светлой/тёмной темы и сохранение пользовательского выбора;
- адаптивное расположение панелей и возможность менять их ширину через сплиттеры.

## Стек
- чистый HTML5 и семантическая разметка;
- современный CSS (CSS custom properties, адаптивная сетка, эффекты);
- нативный JavaScript (без зависимостей) для работы с состоянием матрицы и аналитикой.

## Быстрый старт
1. Склонируйте репозиторий и перейдите в папку проекта.
2. Откройте файл `index.html` напрямую в браузере **или** поднимите статический сервер:
   ```bash
   npx serve .
   ```
3. Задайте размерность матрицы, отметьте параметры и нажмите «Анализировать».

## Структура проекта
```
web-programm/
├── index.html   # каркас интерфейса и шаблон селекторов
├── style.css    # тема, сетка и оформление панелей
├── app.js       # бизнес-логика метода дефицита и генерация отчётов
└── icon.svg     # минималистичный favicon
```

## Руководство по разработке
- Основные функции сгруппированы в `app.js` и снабжены русскими комментариями.
- Стилевой код разделён на тематические блоки с использованием переменных для тёмной/светлой тем.
- Для быстрой проверки изменений удобно использовать расширение VS Code **Live Server** или любую утилиту статического хостинга.
